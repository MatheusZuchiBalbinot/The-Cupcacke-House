# The Cupcacke House

### Como rodar o código:

1 - Extraia o código;

2 - Entre na pasta do código usando cd (Dentro de The-Cupcake-House);

3 - Use no terminal do VScode:
 - npm install
(provavelmente você precisará baixar o Node.js);

4 - Dê o comando:
 - npm run dev
no terminal;

5 - Abra o link que aparecerá no terminal.
